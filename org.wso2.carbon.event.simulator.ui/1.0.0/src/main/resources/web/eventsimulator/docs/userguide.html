Event Simulator

Event Simulator is a tool that can be used to simulate the Event Streams that are predefined.Event Simulator can be used to create and send single event as well as multiple events.

How to use the Event Simulator

1. Start the CEP, log in to the management console , select Tools and click Event Simulator.

Send Single Events

2.Select the Event Stream from the drop down , then enter the attribute value to the generated form


3.Finally click the send button then the Events are sent to any component that this Event Stream is defined such as Event adapter, Event Formatter , Execution plan and etc.

Send Multiple Events

4.Choose a CSV file that contains the event details and upload it

5.Configure the uploaded file for target Event Stream


6.Select the target Event Stream from the drop down list

7.Set the File Delimiter value, Field delimiter is the character which separate the event detail values in an event detail in the file, Most of the times it is “,” comma.

8.Click the configure button

9.After configure the file , click the Play button to create Events and send them to any component that Event stream is defined


This tool can be used to debugging and monitoring the Event Input/Output adapters , Execution plans and Event Formatters.
